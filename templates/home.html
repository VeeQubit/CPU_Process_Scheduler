About The Project

This CPU Scheduling Simulator is built using Flask and JavaScript. It helps students understand operating system scheduling concepts through interactive visualization and performance analysis.

Supported Algorithms

  • First Come First Serve (FCFS)
  • Shortest Remaining Time First (SRTF)
  • Round Robin (RR)
  • Priority Scheduling (Preemptive & Non-preemptive)
